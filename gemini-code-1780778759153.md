# Bilhuset Säffle - Premium Web-plattform

En modern, responsiv och lyxigt animerad webbplats för **Bilhuset Säffle**, optimerad för att hostas kostnadsfritt via GitHub Pages. Designen följer samma minimalistiska och exklusiva formspråk som ledande premiumtillverkare.

## 🚀 Funktioner
* **Premium Mörkt Tema:** Färgkodat efter guld och djup svärta för att ge en känsla av exklusivitet.
* **Flytande JavaScript-animationer:** Skonsam scroll-reveal, räknare som tickar upp live och integrerad preloader.
* **Fordonsgalleri:** Anpassat för personbilar, klassiker och motorbåtar.
* **Fullt Optimerad:** Färdiga SEO-metataggar och Open Graph-stöd för social delning.
